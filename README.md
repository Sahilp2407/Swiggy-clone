
# 🍔 Swiggy Clone – Frontend Project

> A responsive and interactive clone of the Swiggy food delivery platform built using **HTML**, **CSS**, and **JavaScript** – designed to simulate a real-world modern food ordering experience.

## 🧩 Tech Stack

- 🧱 **HTML5** – Semantic structure  
- 🎨 **CSS3** – Responsive design + animations  
- ⚙️ **JavaScript (Vanilla)** – Dynamic DOM, location detection, interaction logic  
- 📍 **OpenStreetMap API** – For reverse geocoding

---

## 📌 Key Features

- 📱 Fully responsive design (mobile + desktop)  
- 📍 Location detection using Geolocation API  
- 🔽 Popular location dropdown with hover selection  
- 🔎 Smart food/restaurant search bar (demo)  
- 🧾 Dynamic restaurant and menu rendering  
- 🛒 Cart system (frontend simulation with +/– quantity)  
- 🎯 Meal subscription plans UI  
- 💫 Smooth UI animations & hover transitions

## 🌀 Bonus: Animation Ideas You Can Add

> Want to make it feel more alive? Add these in your CSS:

```css
/* Button hover bounce */
.button:hover {
  transform: scale(1.05);
  transition: transform 0.3s ease;
}

/* Fade-in section animation */
.section {
  animation: fadeInUp 0.6s ease;
}
@keyframes fadeInUp {
  from {
    transform: translateY(40px);
    opacity: 0;
  }
  to {
    transform: translateY(0);
    opacity: 1;
  }
}
```
